This file describes how to use the corresponding files "Evaluate_Raccoon_Life.py" and "Georges_life.py".

"Evaluate_Raccoon_Life.py" is the name of the processing script to be opened in Python 3.6.

This script requires the INPUT file named "2008Male00006.txt". The processing script accomplishes several tasks:
-opens and reads the input file to the python environment
-converts the data into a dictionary format with headers as dictionary keywords
-computes various summary statistics of the numerical data
-calculates movement distance of the simulated raccoon
-writes various analyses of the data to an OUTPUT file named "Georges_life.txt"

Required input file: 2008Male00006.txt
Created output file: Georges_life.txt
